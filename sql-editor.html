どこで何を実行するか

Supabase の SQL Editor（ブラウザ）

許可されるのは SQL だけです。シェルコマンドは実行できません。

SELECT 1 AS ok;

上の「Supabase の SQL を開く」から同じ画面へ移動し、下書きを貼って Run（⌘↵）します。

ターミナル（この PC）

リポジトリのルート（redlight フォルダ）で、ローカルのスクリプトを実行します。

./scripts/sql-shell.sh

接続先は .env.local の SUPABASE_DATABASE_URL と SUPABASE_DB_PASSWORD（任意）です。
